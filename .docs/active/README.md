# Devys Active Plans

Updated: 2026-04-19

## Purpose

This directory holds active work plans only.

## Rules

- Multiple active plans are allowed.
- Name plans for the slice of work they cover.
- Move a brief here from `../future/` when work actually starts.
- Do not leave closed work here. Delete it or promote the durable parts into `../reference/`.
- Do not use a plan doc as a dumping ground for immutable architecture rules.

## Current State

If this directory only contains `README.md`, no active plan is currently declared.
